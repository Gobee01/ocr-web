### 1. nesting ui_shared repo (Recommended for windows)
    clone ui_shared submodule
    rename ui_shared to shared


### 2. env file
    Copy file .env.prod and create a new file .env in the same directory


### 3. Start App
    Follow step 1
    Execute "npm install"
    Execute "npm start"



Node Version: v16.13.2 or v14
